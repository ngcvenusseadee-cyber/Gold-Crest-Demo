# Lumara Estates

Landing demo para una desarrolladora residencial premium, creada con `Next.js` y `Tailwind CSS`.

## Instalar

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Produccion

```bash
npm run build
npm run start
```

## Notas

- El entorno actual no tiene `node/npm`, por eso el proyecto fue creado manualmente.
- Las imagenes usan URLs remotas de `Unsplash` como apoyo visual para la demo.
- El copy y la identidad estan listos para personalizarse segun el cliente final.
